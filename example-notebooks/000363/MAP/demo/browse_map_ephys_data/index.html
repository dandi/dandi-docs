    Browse datasets on DANDI¶

    This notebook will provide instructions to read a dataset from DANDI and browse the corresponding NWB file. The full set of instructions on NWB can be found here.

    The following packages are necessary to run this notebook. Please install them to your conda environment by typing each of the code blocks in your terminal:

    1. pynwb:

      pip install pynwb

    2. nwbwidgets:

      pip install nwbwidgets

    3. fsspec, requests, and aiohttp

      pip install fsspec requests aiohttp

    This notebook allows you to stream data directly from DANDI to preview the contents of the .nwb file. We currently recommend downloading the .nwb file of interest to your local machine after you explore it with the nwbwidget to ensure optimal performance for downstream analyses.

    If you want to explore the NWB file in streaming mode, you will find instructions at the bottom of the notebook to do so. Please note that converting the units object to a dataframe takes a long time in this mode and it is recommended that you bring objects into memory one at a time for best performance.

    The dataset used in this notebook can be found at https://dandiarchive.org/dandiset/000363?pos=1.

    The units dataframe contains information such as unit_quality, spike_times, and even the CCF location of the electrode for each unit. In the cell below, we'll write a function that returns the x, y and z CCF coordinates for a given unit in the units_df.

    Please Note: x, y, and z are described in the pipeline as follows:

    ccf_x   :  int   # (um)  Left-to-Right (ML axis)
ccf_y   :  int   # (um)  Superior-to-Inferior (DV axis)
ccf_z   :  int   # (um)  Anterior-to-Posterior (AP axis)

    trial_spikes = []
    for time in stim_on_times:
        # Compute spike times relative to stimulus onset
